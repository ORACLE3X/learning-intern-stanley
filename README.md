# Wave8 intern project
This is an intern project for seting up functionality of backend
## Stack

## Main
1. Node
2. Express JS
3. Typescript
4. Prisma

- Package manager: `npm`

## How to install the project
1. Clone the repo and install the dependencies using `npm` install using prisma using `npx`
2. Update the `.env` with appropriate value
3. Use `npm run dev` to run the server
